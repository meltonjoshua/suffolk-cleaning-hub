# Suffolk Cleaning Hub

A modern, responsive website for Suffolk Cleaning Hub - professional cleaning services across Suffolk.

## Overview

This website showcases Suffolk Cleaning Hub's professional cleaning services for residential and commercial properties across Suffolk, including Ipswich, Felixstowe, and Woodbridge.

## Features

- **Responsive Design**: Fully responsive layout that works perfectly on desktop, tablet, and mobile devices
- **Professional Branding**: Clean, modern design with professional color palette
- **Service Showcase**: Detailed information about residential, commercial, end of tenancy, and deep cleaning services
- **Service Areas**: Highlighting coverage across Ipswich, Felixstowe, and Woodbridge
- **Customer Testimonials**: Social proof from satisfied customers
- **Trust Indicators**: Professional credentials and guarantees
- **Accessible Navigation**: User-friendly navigation with mobile-friendly hamburger menu
- **SEO Optimized**: Proper meta tags and semantic HTML structure

## Technology Stack

- **Frontend**: React 18 with Vite
- **Routing**: React Router DOM
- **Styling**: CSS Custom Properties (CSS Variables) with modular CSS files
- **Typography**: Google Fonts (Inter & Playfair Display)
- **Build Tool**: Vite for fast development and optimized builds
- **Linting**: ESLint for code quality

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/meltonjoshua/suffolk-cleaning-hub.git
cd suffolk-cleaning-hub
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Linting

```bash
npm run lint
```
